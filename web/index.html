Advanced options & help

Don't see your board in the popup? Use a USB cable that carries data (not charge-only). On Windows you may need a driver: CH340 (D1 Mini / WiFi Nugget) or CP2102. On Mac/Linux it's usually built in. Then reload and try again.

USB Nugget, Bluetooth Nugget, or Nibble won't connect? Put it in install mode: hold BOOT, tap RESET once, then release BOOT. (No reset button? Hold BOOT while plugging in the USB cable, then release.)

Have a compiled .bin? Drop it here to flash it to any connected ESP board. Uses the speed setting above.

Advanced Leave at 0x0 for a full/merged image; use a specific offset only for a partial image (e.g. an app-only build).

Every firmware image published here is SHA-256 hashed and (once migrated off the imported set) carries a GitHub build-provenance attestation tying it to its source commit and toolchain.

Flasher powered by self-hosted esptool-js. Nothing is uploaded — flashing happens locally between your browser and the board. Need the console? Open the Serial Monitor. Made by Retia LLC.
